## Common Subtitle Errors

**Critical Errors** (High frequency, significant meaning change):
1. **福祉霊感** → 指揮官 (Commander - mojibake corruption)
2. **副司令官** → 指揮官 (Wrong title, different rank)
3. **ラビ** → ラピ (Rapi character name)
4. **アニース** → アニス (Anis character name)
5. **マリア** → マリアン (Marian vs. Maria confusion)

**Moderate Errors** (Occasional, may cause confusion):
6. **政府中央** → 中央政府 (Word order)
7. **地上** ↔ 地表 (Surface/Above ground - context dependent)
8. **核** → コア (Core, technology term)
9. **腐敗** → 腐食 (Corruption, technical term)
10. **女神小隊** → 女神部隊 (Goddess Squad)

**Minor Errors** (Low impact, mostly stylistic):
11. Inconsistent romanization (e.g., "Tetra Line" vs "Tetra-Line")
12. Missing katakana elongation marks (ー)
13. Honorific inconsistencies (-san, -sama)
14. Number format inconsistencies (arabic vs. kanji)

See YOUTUBE_SUBTITLE_CORRECTIONS.md for complete error pattern database with timestamps and correction strategies.

## Naming Conventions

### Character Profile Files
Format: `docs/characters/{CharacterName}_profile.md`
- Use English canonical name
- CamelCase for multi-word names
- Examples: `Rapi_profile.md`, `Scarlet_profile.md`

### RAG Document Files
Format: `docs/videos/NIKKE 第{N}章 {ChapterName} - RAG文档.md`
- Use chapter number and Chinese name
- Example: `NIKKE 第14章 旅行 - RAG文档.md`

### Worldbuilding Files
Format: `docs/worldbuilding/{Topic}.md`
- Use English topic name
- Lowercase with underscores
- Examples: `ark_locations.md`, `three_manufacturers.md`
