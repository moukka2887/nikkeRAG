# 核心概念

## NIMPH（ニンフ）的删除与后果

**NIMPH系统的三大功能**:
1. **意识与记忆控制**: 强制压制创伤与恐惧
2. **纳米机器修复**: 自动修复身体损伤
3. **身份控制**: 定义"英雄"的道德与行为约束

**删除NIMPH的后果**:
- **记忆爆发**: 被压制的创伤记忆涌现
- **身体脆弱**: 失去自动修复，受伤无法快速恢复
- **精神崩溃**: 失去强制麻痺，真实的恐惧浮现
- **身份重构**: 必须重新定义自己是谁
- **神经毒性**: Unchained的毒素副作用加剧痛苦

## Unchained（解除劑）的代价

**表面承诺**:
- 获得对抗Rapture的终极武器
- 可能拯救被NIMPH控制的NIKKE

**真实代价**:
- 永久删除使用者的NIMPH
- 导致严重的神经毒性反应
- 造成无法逆转的心理创伤
- 可能引发智能或道德崩溃

## Hero Complex（英雄病）

**定义**: 强迫自己保持"完美英雄"形象，导致的心理障碍

**Laplace的症状**:
