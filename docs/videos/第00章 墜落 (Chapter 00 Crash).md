# 关键概念

## Rapture (ラプチャー)
- **定义**: 人类的主要敌人，机械生命体
- **特性**:
  - 会被爆炸声和战斗声吸引
  - 对人类武器（小口径）基本免疫
  - 需要 Nikke 的大口径武器才能有效对抗
  - 在地表广泛存在
- **威胁**: 前任指挥官就是因为试图用对人武器攻击 Rapture 而死

## Nikke (ニケ)
- **定义**: 为对抗 Rapture 而制造的战斗用人形单位
- **特性**:
  - 具有自我意识和情感
  - 外表类似人类但本质是机械
  - 会受伤/故障，但人类的医疗物品对其无效
  - 拥有强大的战斗能力
  - 必须由人类指挥官指挥才能发挥最大战力
- **限制**: 需要指挥官的命令才能采取行动

## 指挥系统
- **机关识别码**: 用于验证指挥官身份的系统
- **指挥权转移**: 通过机关识别码可以转移分队的指挥权
- **紧急流程**: 在前任指挥官无法履行职责时，可以跳过常规交接流程
