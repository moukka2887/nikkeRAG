# 章节时间轴

## Part 1: 日常与任务简报 【約96年前】

**场景**: Goddess部队基地
**关键事件**:
- Red Hood听着老歌，与Snow White互动
- Goddess小队成员集合进行作战简报
- 讨论攻击轨道电梯、寻找Queen的计划
- Red Hood的侵蚀状态被揭露

**核心剧情**:
指挥官召集Goddess小队进行简报，说明人类与Rapture的战况。虽然Goddess小队战绩辉煌，但整体战局仍然不利。指挥官提出Queen可能藏身于轨道电梯连接的宇宙空间站，计划派遣约200名量产型Nikke支援进攻。

关键转折：Red Hood的眼睛变红，声音改变，揭示她已被侵蚀。侵蚀是被特殊Rapture触手攻击后的状态，会导致Nikke失控攻击同伴。Red Hood表示这是最后一战，之后将退役回故乡。

## Part 2: 前往Elysion研究所 【約96年前】

**场景**: 前往Elysion第32研究所途中
**关键事件**:
- 队伍前往接收新成员Cinderella（仙履奇缘）
- Elysion第32研究所被Rapture袭击，全员阵亡
- 发现Rapture与Nikke融合的恐怖现象
- Cinderella失踪，专用武装"玻璃鞋"下落不明

**重要发现**:
研究所被约200台Rapture袭击，其中包含4台代号"Ultra"的高级个体。这是针对Fairy Tale模型的有计划袭击。现场发现Rapture将Nikke零件刺入死亡的Nikke体内，呈现融合迹象，显示Rapture的行为模式正在发生不寻常的变化。

## Part 3: Red Hood的离别决定 【約96年前】

**场景**: 基地 - Red Hood与队友的最后时光
**关键事件**:
- Red Hood头痛加剧，侵蚀症状恶化
- 向队友们分发礼物作为告别
- Rapunzel坚持要找到侵蚀的治疗方法
- Snow White请求Red Hood留下来

**感人对话**:

**Rapunzel**: "立ち向かって戦ってください。いつものように。治療法を研究していますから、諦めないでください。"
（请像往常一样战斗。我正在研究治疗方法，请不要放弃。）

**Snow White**: "私の望みはただ1つです。いつものように過ごして、最後の瞬間には私たちと一緒にいてください。故郷には帰らないで。"
（我只有一个愿望。像往常一样度过，在最后时刻请和我们在一起。不要回故乡。）

**Red Hood**: "分かった。そうしてやるよ。スノーの頼みだからな。"
（我知道了。我会那么做的。因为是Snow的请求。）

## Part 4: 轨道电梯作战 【約96年前】

**场景**: 轨道电梯入口
**关键事件**:
- Goddess小队突破Rapture防线
- 遭遇代号Ultra的强敌
- 在宇宙电梯中遭遇变成Anachiro的Cinderella
- Anachiro展现压倒性实力，Goddess小队全灭

**战斗描述**:
队伍成功突破大量Rapture，击败了Ultra。但在轨道电梯中遭遇了被侵蚀并改造的Cinderella，代号Anachiro。她拥有强大的光束武器和防御盾，轻松击败了整个Goddess小队。Lilith严重过热无法行动，其他成员也身受重伤。

## Part 5: Red Hood的离去 【約96年前】

**场景**: 战后 - Red Hood独自离开
**关键事件**:
- Red Hood趁队友重伤之际独自离开
- 与Snow White的痛苦告别
- Snow White哭着说"我会开枪的""我不会哭的"
- Red Hood最终还是离开了

**最痛心的对话**:

**Snow White**: "私が打ちます。打てばいいんでしょう。泣きませんから、表情1つ変えずに打てます。だから行かないでください。"
（我来开枪。开枪就行了吧。我不会哭的，我能面无表情地开枪。所以请不要走。）

**Red Hood**: "分かったよ。" （我知道了。）

Snow最终哭喊着："嘘つき大嫌い！大嫌いだ！レッドフード行かないで！"
（最讨厌骗子！最讨厌了！Red Hood不要走！）

## Part 6: 30年后的世界 【約66年前】

**场景**: 30年后 - 人类败退至Ark
**关键事件**:
- 人类战败，全面撤退至地下Ark避难
- Goddess小队连续作战2个月未休息
- Rapunzel继续研究侵蚀治疗方法
- Snow White制作武器，希望变强以不再失去伙伴

**时代变迁**:
人类在与Rapture的战争中彻底失败，被迫进入地下Ark避难。Goddess小队成为守护Ark入口的主力，2个月内经历了212次袭击。队员们身心疲惫，但仍在坚持。

**角色成长**:
- Rapunzel日夜研究侵蚀治疗法，已掌握基本原理但实施困难
- Snow White不断制作武器，包括为Red Hood设计的超大口径步枪
- Dorothy学会了更灵活的战斗方式
- Grave依然冷静但内心承受巨大压力

## Part 7: 危机降临 【約66年前】

**场景**: Ark入口防卫战
**关键事件**:
- 4支大型Rapture部队同时进攻
- Anachiro随军出动
- Goddess小队在Lilith和指挥官不在时独自应战
- 战况极其危急，面临全灭危机

**绝境描述**:
敌方似乎察觉到Lilith和指挥官不在，发动总攻。Goddess小队虽然奋战，但面对压倒性的数量和Anachiro的再次出现，陷入绝境。正当众人濒临崩溃时，熟悉的枪声响起。

## Part 8: Red Hood归来 【約66年前】

**场景**: 战场 - Red Hood的奇迹归来
**关键事件**:
- Red Hood从后方袭击Rapture部队
- 与Goddess小队重逢
- 5人联手对抗Anachiro
- 击碎Anachiro的"玻璃鞋"，战胜强敌

**重逢时刻**:

听到熟悉的枪声，队员们难以置信：
"この銃声...聞き覚えが..."（这枪声...听着熟悉...）
"レッドフード！本当にレッドフードなのですか？"（Red Hood！真的是Red Hood吗？）
"大丈夫です。レッドフードですから、絶対に負けません。"（没问题的。因为是Red Hood，绝对不会输。）

**终极之战**:
Goddess小队5人再次聚首，展开与Anachiro的决战。通过完美配合，成功击碎Anachiro的"玻璃鞋"核心，取得胜利。

## Part 9: 真正的告别 【約66年前】

**场景**: 战后 - Red Hood与队友的和解
**关键事件**:
- Red Hood侵蚀似乎好转但仍不稳定
- 向队友们道歉并真诚告别
- 给每位队友留下礼物和话语
- Snow White接受了Red Hood的卡带播放器
- Red Hood再次离开，但这次是真正的告别

**Red Hood的礼物**:
- 给Rapunzel：一本书《熱くて盲目的なもの》（热情而盲目之物）
- 给Snow White：卡带播放器（Red Hood的宝物）
- 给其他队友：鼓励和祝福

**最后的告别**:

Red Hood: "みんな、つまんない私を受け入れてくれて、仲間として付き合ってくれてありがとな。お前らと一緒に過ごした時間、一緒に戦った時間、本当に最高の瞬間だった。私は幸せだった。お前らと一緒にいられて幸せだった。"
（谢谢大家接纳无趣的我，作为伙伴与我相处。与你们一起度过的时光，一起战斗的时光，真的是最美好的时刻。我很幸福。能和你们在一起，我很幸福。）

Snow White哭着说："一緒に戦えて...本当に良かった。"
（能一起战斗...真的太好了。）

**场景**: Red Hood归来8年后 - Red Hood遇见量产型Nikke Rapi

- Red Hood倒在废墟中醒来

# 关键概念

## 侵蚀（新色/Corruption）
- **定义**: 被特殊Rapture触手攻击后，Nikke体内的纳米机器被污染，转化为Rapture的思考回路
- **症状**:
  - 眼睛变红
  - 声音改变
  - 头痛
  - 最终会失去记忆和自我，攻击人类
- **发展阶段**:
  - 初期：症状轻微，可用冷冻休眠暂停
  - 中期：Red Hood的状态，能保持意识但身体逐渐失控
  - 末期：完全失控，需要被处决
- **治疗方法**:
  - 官方推荐：脑部破坏（实质上是处决）
  - Rapunzel研究：理论上可移除头部纳米机器，但技术难度极高
  - Red Hood案例：唯一长期保持意识的案例，30年后似乎自然克服
- **象征意义**: 无法逃避的命运、与时间的斗争

## Goddess小队（ゴッデス部隊）
- **成立**: 约2年前
- **成员**: Red Hood, Snow White, Rapunzel, Dorothy, Grave (Lilith)
- **地位**: 人类联合军的王牌部队
- **战绩**: 接近全胜，被称为"勝利の女神"（胜利女神）
- **象征**: 希望、胜利的象征
- **现状（30年后）**: 人类最后的守护者

## Fairy Tale模型（フェアリーテール）
- **第一世代**: Goddess小队成员
- **第二世代**: Cinderella等，投入大量资金和技术
- **特点**:
  - 适应者拥有强大自我和独特能力
  - 单体战斗力远超量产型
  - 需要巨额资金和高端技术
- **Cinderella**: 短期歼灭战特化，配备"玻璃鞋"武装

## Rapture与Nikke的融合
- **首次发现**: Elysion第32研究所
- **现象**: Rapture将零件刺入并融合到Nikke体内
- **目的**: 不明，可能是Rapture的新型攻击方式
- **代表**: Cinderella被改造成Anachiro

## 轨道电梯与Queen
- **理论**: Queen藏身于轨道电梯连接的宇宙空间站
- **证据**:
  - Rapture进攻初期就摧毁了所有发射台
  - 轨道电梯附近集结大量Rapture
- **目标**: 消灭Queen以瓦解Rapture指挥系统

## Ark（方舟）
- **定义**: 人类最后的地下避难所
- **现状**: 30年后，人类全面撤退至此
- **防御**: Goddess小队等Nikke部队守护入口
- **象征**: 人类文明的最后堡垒

## 30年的时间跨度
- **Red Hood**: 沉睡30年，奇迹般克服侵蚀
- **世界**: 人类彻底败退
- **Goddess小队**: 持续战斗，疲惫不堪
- **象征**: 漫长的等待、坚持与希望

# 核心主题分析

## 1. 牺牲与选择
Red Hood选择独自离开，承受孤独的死亡，而不是让队友承受杀死她的痛苦。这体现了她的自我牺牲精神和对伙伴的深厚情感。

## 2. 羁绊与离别
Goddess小队成员之间的深厚羁绊是故事的核心。Snow White的崩溃、Rapunzel的承诺、Dorothy的后悔，都展现了离别的痛苦。

## 3. 时间与等待
