# NIKKE主线剧情时间推断分析（基于事件细节）

## 说明
由于主线剧情**没有明确标注章节之间的时间间隔**，本文档通过统计**具体事件**来推断实际时间流逝：
- 战斗场景数量与描述
- 睡眠/休息/野营次数
- 日夜循环标记
- 疲劳状态与补给需求
- 明确的时间描述（"明日早朝5时"等）

## 结论

### 基于事件细节的时间推断（含通勤与昏迷）

**主线20章实际时间跨度：2-2.5个月**

**依据**：
1. ✅ 至少8-12次睡眠/野营场景
2. ✅ 明确"明日早朝5时"（过夜等待）
3. ✅ 明确"1週間"旅程（第26章）
4. ✅ 多处"数日"标记（3-5天间隔）
5. ⚔️ 密集战斗但有休息间隔
6. 💤 **昏迷/治疗时间**：至少12-24小时总计
7. 🔧 **维护/修理时间**：至少1.5-2天总计
8. 🚶 **通勤/移动时间**：至少10-15天总计

**时间成本分解**：
- **战斗时间**：15-20天（密集战斗）
- **休息睡眠**：8-12天（每晚6-8小时）
- **通勤移动**：10-15天（含1周长途旅程）
- **维护修理**：1.5-2天（砲台修理、Nikke维护）
- **昏迷治疗**：1-2天（多次受伤）

### 时间"隐藏成本"分析

**每天实际行动时间**（人类在队时）：
- 有效行动：8-10小时（战斗、移动、作业）
- 休息睡眠：6-8小时

- 其他准备：2-3小时

**例子**：第21章砲台修理

- 但实际耗时：**1整天**（包含移动、休息、准备）

- 战斗时间：3-4小时
- 但实际耗时：**2天**（发现→回基地→过夜→明日5时→战斗→治疗）
