## 校对策略

### 阶段1：系统性错误扫描（自动化）

使用正则表达式批量查找常见错误模式。

#### 1.1 同音异字错误
```bash
# 可疑的色情词汇（通常是"生死"误识别）
grep -rn "精子|精液|性交|做爱|阴茎|阴道" docs/videos/

# 副司令官误识别
grep -rn "福祉霊感|福祉令官|服従令官|複製令官" docs/videos/

# 复仇误识别
grep -rn "復習する.*復讐|復習.*仇" docs/videos/

# 濒死误识别
grep -rn "節減.*死|節減.*境" docs/videos/

# 其他常见同音词
grep -rn "腹筋.*復帰|福祉.*副司|生死.*精子" docs/videos/
```

**预计发现**：10-30处错误

#### 1.2 人名错误
```bash
# Brother误识别
grep -rn "ブラゾン|ブザー(?!フ)" docs/videos/

# 其他人名混淆
grep -rn "ラビー|ラッピー|モダーニャ" docs/videos/
```

**预计发现**：5-15处错误

#### 1.3 组织名错误
```bash
# Ark相关
grep -rn "アーク|アック|アーグ" docs/videos/

# Missilis相关
grep -rn "ミシリス|ミスリス|ミツリス" docs/videos/
```

**预计发现**：0-10处错误（较少）
